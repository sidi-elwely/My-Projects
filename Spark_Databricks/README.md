Project Overview

This project focuses on building a data processing pipeline using Apache Spark on Databricks to analyze ball-by-ball data from all Indian Premier League (IPL) matches up to the 2017 season (637 matches).

The goal is to transform raw sports data into structured and analytics-ready datasets, following a Data Engineering approach: ingestion, transformation, and data modeling in a cloud environment.

Tech Stack

Apache Spark (PySpark)

Spark SQL
